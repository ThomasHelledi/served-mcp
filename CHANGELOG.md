# Changelog

All notable changes to Served.MCP will be documented here.

## [2026.2.1] - 2026-02-01

### Added
- Public documentation repository (docs-only)
- NON-AI-MIT license to protect against AI training
- Comprehensive tool reference for 40+ MCP tools
- Configuration and getting started guides
- Common workflow examples

### Changed
- Source code moved to private Forge repository
- Documentation separated from source for public access

### Security
- Added robots.txt blocking AI crawlers (GPTBot, CCBot, etc.)

## [2026.1.3] - 2026-01-29

### Added
- UnifiedHQ integration - all endpoints now use apis.unifiedhq.ai
- Forge Analytics - built-in tracing with dashboard support
- Fork support - GitHub Actions workflow for forks

## [2026.1.2] - 2026-01-17

### Added
- Unified file format for documentation
- SDK tracing integration
- OpenTelemetry support for tool calls
- Extended DevOps tools

## [2026.1.1] - Initial Release

### Added
- Initial MCP server implementation
- 40+ tools for Served platform access
